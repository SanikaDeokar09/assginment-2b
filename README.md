## Procedure:

1) Create a new repository on GitHub.com
- Login to GitHub
- Click New Repository
- Enter repository name → assginment-2b
- Select Public
- Do not initialize with README, license, or .gitignore
- Click Create Repository

2) Open Terminal (Git Bash / PowerShell)
- Go to your project folder
- Right click → Open in Terminal

3) Change the current working directory
cd C:\Users\Sanika\OneDrive\Desktop\wad\ass1b

4) Initialize Git
git init -b main

5) Add files
git add .

6) Commit files
git commit -m "First commit"

7) Copy repository URL
https://github.com/SanikaDeokar09/assginment-2b.git

8) Add remote

9) Push code
